This program is made to buy books. It reads a small .csv database, and allow the user to search 
for books and check them out by writing the selected books to a .txt file. There are 3 menus, 
MainMenu, CartMenu, SearchMenu, and they all can connect to one another, and present the user 
specific options in each. 

>> Download the files from phase_2 
>> Open cmd
>> Use "cd" to change directory to the folder where the items are downloaded
    ex. "cd c:\users\User\downloads\thisfolder"
>> Run "javac Main.java"
>> Run "java Main"
>> Follow the options of the program
